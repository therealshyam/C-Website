Follow these simple steps to customize Entheos templates.

  • Step 1: Defining the site in Dreamweaver
  • Step 2: Customizing the Header with your Company Name
  • Step 3: Customizing Links
  • Step 4: Customizing the footer with your company details
  • Step 5: Adding new content area pages
  • Step 6: Changing the carousel images

You'll also find a wide variety of web design tips and tricks in our Web Design Guide.

Step 1: Defining the site in Dreamweaver

  • Define your site in Dreamweaver by clicking on Site/New Site.
  • Fill in your site details in the Site Definition panel. Enter your site name and use the folder icon to browse to the folder where your site files reside on your computer. Once you have defined your site, click on Done in the Site Definition panel.
  • You will now see the below given list of folders and files in your local site:
    1. copyright.html - This file gives the copyright information for using this template. Please read the terms and proceed to customzie then template.
    2. images folder - Contains the images used in the site. All updated images should be exported to this folder.
    3. css/style.css, css/bootstrap-responsive.css and css/bootstrap.css - These are the external style sheet used in the template. The styles created will be used to format text throughout the site. Additions and modifications to existing styles can be made here.
    4. js/jquery.js, js/bootstrap.js, js/respond.js and js/bootstrap-typeahead.js - These are the external javascript functions used in the template. The styles created will be used to format text throughout the site.
    5. index.html - The multiple content areas are loaded in this page at the beginning and later allow the users to navigate between the content areas.
    6. help_files.html - Guide to customizing Entheos templates.

Step 2: Customizing the Header with your Company Name

  • Change the site name using the editor like Adobe Dreamweaver, etc. You can also replace the font type, size and color in the style sheet. You can adjust the text size if the name is too long else adjust the div size in style.css file.

Step 3: Customizing Links

The template have css based links
  1. Open index.html using Notepad or an HTML editor like Dreamweaver.
  2. Now you can select and edit the text in the navigational links. And also you can create more navigational links here.
  3. Save the changes.

Deleting extra buttons or links

If you'd like to delete unwanted buttons or links, select the button or link and delete it.

Step 6: Changing the carousel images

  • You can also change the carousel image of this template.

  • Scale all your images to the same height. And also the images should be in .jpg format.
